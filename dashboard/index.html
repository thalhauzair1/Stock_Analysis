⚡ Momentum (MOM) Weight 20%

What it is: Measures how aggressively the stock moved today compared to the S&P 500. A stock that's up 8% on a flat market day is doing something special.

daily_pct = (close − prev_close) / prev_close × 100 rel_strength = daily_pct − SPY_daily_pct score = clamp(daily_pct / 10.0, 0, 1) score += 0.10 if rel_strength > 5% (bonus: beating SPY hard)

Score scale:

0–flat → 0.0 +5% move → 0.5 +10% move → 1.0
💡 Best combined with high RVOL — a big move on huge volume is institutional, not retail noise.
📊 Volume / RVOL (VOL) Weight 20%

What it is: Relative Volume (RVOL) compares today's trading volume to the 20-day average. Unusually high volume means institutions and funds are actively moving in — that's the fuel behind real breakouts.

avg_vol_20d = mean(volume[-21:-1]) (excludes today) RVOL = current_volume / avg_vol_20d score = clamp((RVOL − 1.0) / 2.0, 0, 1)

Score scale:

RVOL 1x → 0.0 (average day) RVOL 2x → 0.5 (elevated) RVOL ≥3x → 1.0 (very strong)
💡 Volume alone doesn't mean bullish — it means something is happening. Always pair RVOL with price direction.
📐 Technical — RSI + MA50 (TECH) Weight 15%

What it is: Combines two classic indicators into one score. RSI tells you if the stock is overbought or oversold. MA50 tells you if the broader trend is up or down.

RSI (14-day Relative Strength Index)

delta = daily price changes gain = positive deltas, loss = negative deltas avg_gain = EWM(gain, span=14) avg_loss = EWM(loss, span=14) RSI = 100 − 100 / (1 + avg_gain / avg_loss)
<30 oversold → 0.2 30–49 → 0.4–0.7 50–70 sweet spot → 1.0 70–80 overbought → 0.3 >80 filtered out ✕

MA50 (50-day Simple Moving Average)

MA50 = mean(close[-50:]) score = 1.0 if close > MA50 (uptrend) = 0.0 if close ≤ MA50 (downtrend)

ATR (Average True Range) — used in prediction, not scored directly.

TR = max(H−L, |H−prev_close|, |L−prev_close|) ATR14 = EWM(TR, span=14) ← daily volatility in dollars
TECH score = 0.5 × RSI_score + 0.5 × MA50_score
💡 RSI 55–65 + above MA50 is the ideal setup: trending up but not overextended.
⚖ VWAP — Volume Weighted Average Price Weight 10%

What it is: VWAP is the average price of a stock weighted by volume throughout the day — it's where most money actually traded. Institutions use it as a benchmark; price above VWAP means buyers are in control.

typical_price = (High + Low + Close) / 3 VWAP = Σ(typical_price × volume) / Σ(volume) pct_vs_vwap = (close − VWAP) / VWAP × 100 score = clamp(0.5 + pct_vs_vwap / 10, 0, 1)
5% below VWAP → 0.0 At VWAP → 0.5 5% above VWAP → 1.0
💡 Reclaiming VWAP after dipping below is a strong intraday reversal signal used by day traders and algos.
🚀 Breakout (BRK) — 20-day High + Volume Confirmation Weight 15%

What it is: Detects when a stock breaks above its 20-day high. A breakout confirmed with above-average volume is much more reliable — the move has conviction.

high_20d = max(close[-21:-1]) (excludes today) is_breakout = close > high_20d is_vol_conf = current_volume > avg_vol_20d × 1.5 breakout_pct = (close − high_20d) / high_20d × 100
No breakout → 0.0 Within 2% of high → 0.3+ Price break, low vol → 0.5 Confirmed (price + vol) → 0.75–1.0
💡 The 20-day high is a widely-watched resistance level. When it breaks with volume, it often triggers stops and momentum buying — a self-fulfilling signal.
📰 News Sentiment Weight 10% needs NewsAPI key

What it is: Fetches recent news headlines via NewsAPI and scores sentiment. The system automatically uses the best available method — Claude (Anthropic) if an Anthropic key is set, GPT-4o-mini if an OpenAI key is set, otherwise fast keyword matching.

Mode 1 — 🤖 Claude (claude-haiku-4-5) active when ANTHROPIC_API_KEY is set

Headlines → Claude Haiku → JSON response: { "overall": 0.82, ← weighted sentiment score "sentiments": [0.9, 0.7, …], ← per-headline scores "catalysts": ["earnings beat", "guidance raised"] } Claude understands nuance: "miss but guidance raised" → mildly positive (not just negative) "beat but outlook cut" → mildly negative (not just positive) System prompt is prompt-cached across all 30 ticker calls — saves cost & latency.

Cost: ~$0.001 per full analysis run (30 stocks × 10 headlines each). Results are cached for 1 hour per ticker.

Mode 2 — 🤖 OpenAI (GPT-4o-mini) fallback when OPENAI_API_KEY is set and no Anthropic key

Same JSON format as Claude mode above.

Mode 3 — Keyword Matching fallback — no API key needed

Positive: beat, record, growth, upgrade, buy, profit, revenue … Negative: miss, loss, downgrade, sell, lawsuit, fraud, recall … score = pos_keywords / (pos + neg) + catalyst_boost
💡 Keys go in your .env file: ANTHROPIC_API_KEY=sk-ant-..., OPENAI_API_KEY=sk-..., and NEWS_API_KEY=.... All are optional — the system degrades gracefully without them.
🏦 Smart Money — Institutional & Congressional Trades Weight 10% needs Quiver Quant key

What it is: Tracks filings from US congressional members, dark pool activity, and institutional purchases via the Quiver Quant API. Congress members historically outperform the market because they often have superior information.

Data sources: • Congressional trades (STOCK Act filings) • Institutional 13F filings • Dark pool block trades score = 1.0 if recent insider/congress BUY = 0.5 if no recent activity (neutral) = 0.1 if recent SELL
💡 Set QUIVER_API_KEY in your .env file. Congressional buy disclosures are public law — this is perfectly legal to track and trade on.
🔮 Price Prediction — How the Forecast Works

Data source: 1 year of daily OHLCV (Open/High/Low/Close/Volume) from Yahoo Finance, free and real-time.

Step 1 — Trend (Linear Regression)

Fits a straight line through the last 63 trading days (~3 months) of closing prices. The slope tells you the trend direction and speed.

x = [0, 1, 2, … 62] (day index) y = closing prices (last 63 days) slope, intercept = polyfit(x, y, degree=1) forecast[t] = slope × (63 + t) + intercept

Step 2 — Confidence Bands (ATR × √t)

The shaded area on the chart grows wider over time. This uses the random walk principle: uncertainty scales with the square root of time, just like a coin flip — the further ahead you look, the wider the range of outcomes.

ATR = 14-day Average True Range (daily $ volatility) upper[t] = forecast[t] + 1.96 × ATR × √t lower[t] = forecast[t] − 1.96 × ATR × √t → 95% of outcomes should fall inside this band (assuming no structural change in volatility)

Step 3 — Bull / Base / Bear Targets

Base = end of regression line (where trend points) Bull = max(63d_high × 1.08, base × 1.05) Bear = min(MA50 × 0.92, base × 0.92)

What the chart shows

— Blue line: 6 months of actual history – – Yellow: projected trend (base forecast) ▓ Shaded band: 95% confidence interval (widens with time)
⚠ Important: This is a statistical trend projection, not a guarantee. Linear regression assumes the recent trend continues. It cannot predict earnings surprises, macro shocks, or news events. Always combine with the signal scores — a strong forecast on a high-scoring stock is much more meaningful than one on a weak setup.
